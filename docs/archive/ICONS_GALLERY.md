## 💡 使用技巧

### 1. 按需导入
```tsx
// ✅ 推荐 - 只导入需要的图标
import { MedicationIcon, CheckIcon } from '@/components/icons/medical-icons';

// ❌ 避免 - 导入全部（增加包体积）
import * as Icons from '@/components/icons/medical-icons';
```

### 2. 统一尺寸
```tsx
// ✅ 推荐 - 同一界面使用统一尺寸
<MedicationIcon size="lg" />
<PrescriptionIcon size="lg" />

// ❌ 避免 - 混用尺寸
<MedicationIcon size="sm" />
<PrescriptionIcon size="xl" />
```

### 3. 合理用色
```tsx
// ✅ 推荐 - 符合语义
<CheckIcon variant="success" />
<AlertIcon variant="warning" />

// ❌ 避免 - 颜色冲突
<CheckIcon variant="error" />
<AlertIcon variant="success" />
```

## 🎯 最佳实践

### ✅ 推荐
1. 使用语义化的颜色（成功用绿色，警告用橙色）
2. 保持同一页面图标尺寸一致
3. 为图标添加适当的间距（gap-2, gap-3）
4. 按钮图标使用 md 尺寸
5. 标题图标使用 lg 或 xl 尺寸

### ❌ 避免
1. 避免在浅色背景使用浅色图标
2. 避免过度使用动画效果
3. 避免混用不同风格的图标
4. 避免使用过小的尺寸（< sm）
